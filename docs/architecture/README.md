# Архитектура проекта

Этот каталог содержит архитектурные документы по системе в формате "один модуль — один файл".

## Структура

- `00-system-overview.md` — общая архитектура, принципы и сквозные потоки.
- `modules/frontend-admin.md` — архитектура админского UI.
- `modules/core-api.md` — архитектура Core API.
- `modules/worker.md` — архитектура download/convert worker.
- `modules/queue-redis.md` — архитектура очередей и Redis.
- `modules/storage.md` — архитектура локального хранения.
- `modules/torrent-client.md` — архитектура интеграции с torrent-клиентом.
- `modules/deployment-compose.md` — deployment-архитектура на Docker Compose.

## Единый шаблон модулей

Все модульные документы следуют шаблону из `modules/_module-template.md`:

1. Назначение
2. Границы ответственности
3. Интерфейсы и зависимости
4. Контракты данных
5. Отказоустойчивость
6. Наблюдаемость и безопасность
7. Точки расширения под сериалы
