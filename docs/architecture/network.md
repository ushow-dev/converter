# Сетевая архитектура

## Принцип

Внутренние серверы **никогда не доступны пользователю напрямую**. Все публичные запросы идут через заменяемые Edge proxy. При блокировке Edge — поднимается новый VPS или меняется домен. Внутренняя инфра не трогается.

```
Пользователь → Edge Proxy (заменяемый) → Внутренний сервер (постоянный)
```

Аудитория: Бангладеш, Индия. Edge серверы в Сингапуре (минимальная задержка).

## Внешний слой (Active-Active, Сингапур)

Два Edge proxy, разделённые по типу трафика. Разные домены — разный риск блокировки. При блокировке одного — второй продолжает работать.

| Edge | IP | Что проксирует | Трафик |
|---|---|---|---|
| **Edge 1** | 67.159.52.120 | Сайт (ultrashow.fun) + API proxy | Лёгкий (JSON, HTML) |
| **Edge 2** | TBD | HLS потоки + Player UI | Тяжёлый (видео .ts сегменты) |

**Почему Active-Active:**
- Блокировка stream-домена (жалоба правообладателя) вероятнее чем блокировка сайта
- Сайт работает при блокировке стриминга и наоборот
- HLS не забивает канал сервера с сайтом

## Схема маршрутизации

```
ultrashow.fun (Edge 1 — 67.159.52.120)
    ├── /api/*     → API (178.104.100.36:8000)
    └── сайт       — статика + админка фильмов

{stream-домен} (Edge 2 — новый IP)
    ├── /           → Player UI (45.134.174.84:3100 → потом MinIO)
    └── /media/*    → MinIO (178.63.205.179:9000)
                      └── nginx proxy_cache для .ts (кеш на Edge)

Пользователь знает: ultrashow.fun, {stream-домен}
Пользователь НЕ знает: 178.104.100.36, 178.63.205.179, 45.134.174.84
```

## Действия при блокировке

### Заблокировали stream-домен
1. Привязать резервный домен к Edge 2 IP (5 мин)
2. Обновить `MEDIA_BASE_URL` в конфиге плеера
3. Сайт (ultrashow.fun) продолжает работать

### Заблокировали ultrashow.fun
1. Привязать резервный домен к Edge 1 IP (5 мин)
2. Стриминг продолжает работать

### Заблокировали IP Edge сервера
1. Поднять новый VPS в Сингапуре (~10 мин)
2. Перенаправить домены на новый IP
3. Добавить новый IP в фаервол внутренних серверов

**Внутренняя инфра не трогается ни в одном сценарии.**
